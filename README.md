# 🍽️ Mealify – Restaurant Website (Frontend)

**Mealify** is a fully responsive restaurant website built using **HTML** and **CSS**. It is a frontend-only project that showcases a clean and modern layout for a restaurant business.

---

## 🔍 Description

This project provides an elegant and informative web page for a restaurant. It is designed to be user-friendly and visually appealing, highlighting the restaurant’s identity and services.

### 📌 Sections Included:

- **🏠 Home** – Welcome section with branding and call-to-action.
- **👨‍🍳 Chefs** – A showcase of the team of professional chefs.
- **🍲 Gallery** – A visual collection of meals and restaurant photos.
- **📞 Contact** – A section with a contact form, address, email, phone number, and embedded Google Maps location.

---

## 🛠️ Technologies Used

- **HTML5**
- **CSS3** (Flexbox, transitions, responsiveness)

---

## ✅ Features

- Mobile-friendly responsive design
- Interactive sections with hover effects
- Clean layout using Flexbox
- Contact form and location embedding
- Smooth navigation experience
